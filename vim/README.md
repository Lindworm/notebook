#Vim Pugins

##Vundle

###Install Git

```
sudo apt-get install git
```

###Install Vundle

```
$git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

###Config Bundle

```
cd ~;wget https://raw.githubusercontent.com/liushoukai/notebook/master/vim/.vimrc
```

###Install Bundle

```
$vim +BundleInstall +qall
```

###Frequently Used

```
:PluginInstall
:PluginUpdate
:PluginClean
:PluginSearch
```
